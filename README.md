1.右上のcodeからDowload zipでフォルダをダウンロードしてください。

2.ダウンロードしたフォルダを開いて中身に

  - PHP応用MVC作成.pdf
  - sql_basic.sql
  - SQL基礎問題.pdf

3.中身が正しいことを確認したらPHP応用.pdfの内容に沿ってカリキュラムを進めてください。
